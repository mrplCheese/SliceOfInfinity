This was a project I led at Harvestworks as an intern, several interns got involved!

It takes in a matrix of color values (ARGB) and feeds it through various sound-encoders: 
1. Wavetable synth generator (Esme Nadeau)
2. Conway's Game Of Life --> Sound (Kei Watanabe)
3. 1D bezier spline ring in 6D space --> multichannel, spatial audio output (Micah Lewis)

The user interface includes a 4 "modes"
1. Random value generator + "unsmoothed" data
2. Random value generator + "smoothed" data (2D bezier smoothing before the 6D spatialization)
3. Camera + unsmoothed data
4. Camera + smoothed data

This was designed for an installation that displayed at HarvestWorks in the Summer of 2025.
